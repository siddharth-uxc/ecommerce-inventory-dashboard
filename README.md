# 🛒 Zepto Inventory Dashboard

An interactive Inventory Management and Business Analytics Dashboard built using Python, Pandas, Plotly, and Streamlit.

## 🚀 Features

- 📦 Inventory Tracking
- 📊 Business Analytics
- 💰 Revenue Analysis
- 🔍 Product Search
- 📂 Category Filtering
- ⚠️ Low Stock Monitoring
- 🔥 Correlation Heatmap
- 📈 Interactive Charts
- ⬇️ Inventory Data Download

## 🛠️ Tech Stack

- Python
- Pandas
- Streamlit
- Plotly Express
- Git & GitHub

## 📊 Dataset

Zepto Product Inventory Dataset

Contains:
- Product Name
- Category
- MRP
- Discount Percentage
- Selling Price
- Quantity
- Stock Status
